## Getting Started

### Installing

* Git clone the repo

 ```
git clone git@github.com:a-poco/fileupload.git
```

* Install dependencies

```
bundle install
```

* Migrate the database schema

```
rails db:migrate
```

* Run the server

 ```
rails s
```

* Navigate to localhost:3000
